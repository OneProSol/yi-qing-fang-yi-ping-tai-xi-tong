### 一、相关技术
- 前端：`Vue`、`HTML` / `CSS` / `Javascript`。
- 后端：`Springboot`等。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
### 三、部署所需环境

- 后端：`jdk1.8`。
- 前端：`node`、`vue-cli` 、`npm`  。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示
系统分为：用户、管理员。

用户功能：1. 登录、2. 首页、4. 实时疫情状态、5. 历史出行数据管理、6. 每日健康打卡、7. 外出报备管理、8. 复工申请管理、9. 外出报备审核、10. 复工申请审核、11. 疫情公告信息


管理员功能：1. 登录、2. 首页、3. 管理员列表、4. 角色管理、5. 菜单管理、6. 实时疫情状态、7. 历史出行数据管理、8. 每日健康打卡、9. 外出报备管理、10. 复工申请管理、11. 外出报备审核、12. 复工申请审核、13. 疫情公告信息
